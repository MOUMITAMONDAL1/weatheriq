
 🌦️ Weather Forecast Web App

A stylish and intelligent weather forecast app built with **Streamlit** and **Machine Learning**. It displays live weather using **OpenWeatherMap API**, predicts **rain**, and forecasts **temperature and humidity** trends using historical data.

---

🌟 Features

✅ Real-time weather data for any city  
✅ Predict rain for tomorrow using ML classification  
✅ Forecast next 5 hours' temperature & humidity using regression  
✅ Beautiful UI with dynamic background images  
✅ Supports CSV-based historical data training
